In this elective we've used a few tools that have come in handy for backend web-development

Express

Express is a Node.js web application framework that provides a set of features for web applications.
It is fast, robust and asynchronous. Because of this, it is the leading backend framework for Node

To install express you will require a version of Node and npm. You can install this from the node.js website

In your project repository you will need to create a file called package.json with one simple line of code:

{}

With node installed and package.json created you will be able to do a simple terminal command to install express into your project
npm install express

Express will now be installed and you can proceed to start your first express project.
Below will be an example of a simple web-application using nothing but express:

const express = require("express"); //import express with require
const app = express(); //Inititalize express and save it to a const

//Create a endpoint with parameters request & response
app.get("/", (req, res) =>{
    res.send("Hello World");    //Send a string to the web application
})

//Open the server on chosen port
app.listen(8080, () =>{ 
    console.log("Server is running on port ", 8080);
});
