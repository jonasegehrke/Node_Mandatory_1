Javascript Basics

Description

This page will include some basic but essential knowledge from javascript.

Data Types

Javascript types are dynamic. This means that the same variable can be used to store different data types:

let x; // x is undefined
x = 5; // x is Number
x = "Jonas"; // x is String
x = true; // x is Boolean
const animals = ["Cat", "Tiger", "Dog"]; // animals is Array
const cat = {breed: "American Bobtail Cat", coat: "Medium", lifespan: "13-15 years"}; // cat is Object

Hoisting

Hoisting is JavaScript's default behavior of moving all declarations to the top of the current scope (to the top of the current script or the current function).

Loops

Loops in javascript can be done with for & while but this is commonly slow as you have to itterate through each element of the collection.
That means that if you dont need the index of each element you can simply use map or other functions.
Below will be examples of a few ways to find a element in a collection using loops/map

const carBatteries = [0.32423, 0.65765, 0.3242, 0.13543]; // declaring an Array
const chargedCarBatteries = carBatteries.map(x => x+0.4); // Adds 0.4 to each element (battery) using map
carBatteries.forEach(function(value, index){console.log(`${value} : ${index}`)}); // prints out each battery and index using forEach loop
const drainedBatteries = carBatteries.filter(x => x < 0.5) // Stores batteries with low battery in a seperate array using filter

const movies = [{id: 1, title: "Avengers: Infinity War"}, {id: 2, title: "The Incredible Hulk"}] // declaring array of objects
const foundMovie = movies.find(movie => movie.id === 2; // Finds the object where the id === 2 using find function
const foundMovieIndex = movies.findIndex(movie => movie.id === 2; //Finds the index of the object with id === 2 using findIndex

Functions

A Javascript function is a block of code designed to do a particular task.
The syntax for a javascript function looks like this:

function name(parameter1, parameter2, parameter3) {
    // code to be executed
}

Callback functions

A Callback function is a function set to be excecuted after another function has finsished execution.
We need to use callback functions as many javascript actions are asynchronous, which means that they dont stop the program from running anything else until they are completed.
An example for using callback functions could look like below. This example is taken from express.

app.listen(process.env.PORT || 3000, () =>){ //This sets the port and then calls a callback function
    console.log("Express server listening on port %d in %s mode", this.address().port, app.settings.env); // This  console.log is executed after the server is live
});

Spread Operator

The JavaScript spread operator (...) allows us to quickly copy all or part of an existing array or object into another array or object.
An example would be in a HTTP patch request, as we dont nececarely want to edit alle parts of the data.
Using the spread operator you can combine two objects without overwriting non-shared keys

app.patch("/movies/:id", (req, res) =>{
    const foundMovie = movies.find(movie => movie.id === Number(req.params.id));
    // Using the spread operator below we can combine the two datasets and only overwrite existing key, value
    foundMovie ? movies[foundMovie.id-1] = { ...foundMovie, ...req.body, id:foundMovie.id } : res.sendStatus(404).send({});
    res.send({})
})

Array Methods

Array comes with alot of different methods, below i will show a few:

const diet = ["cucumber", "tomato", "Salad"]; // declaring an array
    diet.push("pizza") // add a new element to array
