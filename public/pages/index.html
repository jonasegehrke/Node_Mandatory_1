Dat-Elective: NodeJS (Spring 2022)

Description of elective

Learn to set up a Node.js server and create APIs.
We will work towards 2 big projects.
The elective aims to make students understand the full stack and the classic separation of backend and frontend.
We will work with fetching data from both a relational and a document databases.
We will also learn how to use external services and libraries some of which include sockets. We may include a web library / framework.
Students are also expected to get comfortable with the command line during the course.

Learning goals

The goal that the student gains knowledge of:

  • Setting up a server with Node.js
  • Working with databases in Node.js.
  • Using package managers: npm
  • Using the terminal.
  • Using a version control system, Git.
  • Building a larger and scalable project.

The goal is that the student is able to use the tools listed above to develop single-page web applications.

Competencies

The goal is that the student is able to use the listed tools and analyse and select a suitable tool for a given task within a web development context.
The objective is to enable the students to understand the backend / frontend separation and take full-responsibility of developing a full-stack application.
